# Quick start

Clone the repository:
```
$ git clone --recursive git@gitlab.com:dalton/dalton.git
```

Build the code:
```
$ cd dalton
$ ./setup [--help]
$ cd build
$ make [-j4]
```

Run the test set:
```
$ ctest [-j4]
```
